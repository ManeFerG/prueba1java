# API Gateway

## Descripción
Este componente actúa como punto de entrada al sistema, gestionando las solicitudes del cliente y redirigiéndolas a los microservicios correspondientes.

---

## Seguridad
Autenticación mediante JWT
Comunicación segura mediante HTTPS

---

## Rutas

/usuarios → usuarios-service
/pedidos → pedidos-service

---

## Configuración
Puerto: 8080

---

## Rol dentro de la arquitectura
Centraliza el acceso a los microservicios, mejorando la seguridad y el control del sistema.

---

## Arquitectura
Forma parte de una arquitectura basada en microservicios desplegada en la nube.
